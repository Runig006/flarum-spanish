## About The Project

Spanish language pack for Flarum Software, includes its extensions and libraries in two Spanish language registers; **Formal Spanish** and **Informal Spanish**, both selectable from the extension administrator dashboard.

This extension is maintained by [@jslirola]https://flarum.es/u/jslirola and [@Darkle]https://flarum.es/u/Darkle members of the verified [Flarum en Espa√±ol](https://flarum.es) community.

## Getting Started

In order to install the Spanish language extension package, you will need to follow these simple steps.


### ‚ö†Ô∏è Minimum Requirements

   ```
flarum/core: "^1.2.0"
   ```

### üì• Installation

Flarum uses Composer to manage its dependencies and extensions. The Spanish language pack is available on <a href="https://packagist.org/packages/flarum-lang/spanish">Packagist</a> and can be managed that way. Make sure <a href="https://getcomposer.org/">Composer</a> is installed on your server, then run the following command at the location where Flarum is installed:
   ```
   composer require flarum-lang/spanish
   ```

### ‚ôª Updating

As with the installation procedure, make sure <a href="https://getcomposer.org/">Composer</a> is installed on your server, and then run the following commands at the location where Flarum is installed:
   ```
   composer update flarum-lang/spanish
   php flarum cache:clear
   ```

## Usage

Once the extension is installed, you will have to activate it from the administrator panel:

![](https://i.ibb.co/7vqYZyf/Captura-de-pantalla-2021-04-09-012031.png)

Next you will be able to select the Spanish register (formal and informal) that you want to use.

![](https://i.ibb.co/wWH6ZQd/Captura-de-pantalla-2021-04-09-012548.png)

üö® Don't forget after selecting the registry to **save changes** and **clear cache**.

If you are not a Spanish speaker and you don't know which register to choose, here is a note to help you.

**NOTE:** On the internet we normally use an informal register so it is advisable to use it as it will be familiar to users. In professional or more serious contexts, choose the formal register.
